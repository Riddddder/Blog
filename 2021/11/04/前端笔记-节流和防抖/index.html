前端笔记：节流和防抖

发表于 2021-11-04 分类于 笔记 本文字数： 1.6k 阅读时长 ≈ 1 分钟

# 区别

节流: n 秒内只运行一次，若在 n 秒内重复触发，只有一次生效
防抖: n 秒后在执行该事件，若在 n 秒内被重复触发，则重新计时

# 应用场景

节流：滚动条监听、搜索框输入
防抖：输入框搜索、窗口大小 resize
