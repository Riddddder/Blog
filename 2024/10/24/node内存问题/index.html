# 问题描述

  • 在一次使用 vue-element-plus-admin 打包时，出现如下报错:

# 问题原因

  • node 内存泄漏耗尽，JavaScript 堆内存不足。因为 Node 是基于 V8 引擎，在 Node 中通过 JavaScript 使用内存时只能使用部分内存。

# 解决方法

  • 安装依赖工具
  • 修改 package.json 如下:[LIMIT 大小自己设置]
  • 最后执行就可以打包啦
